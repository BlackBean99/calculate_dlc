# 전공 E반 3team [calculate_dlc]
### 1. 프로젝트 주제
   IT 분야의 급격한 발전이 세상을 전에 없이 신속하고 편리하게 변화시키는 현재, 컴퓨터 기술의 근본을 재조명하기 위해 본 프로젝트를 기획하였습니다.
   세계에 존재하는 다양한 기술들이 결국은 수학적 연산에 근거한다는 근본적인 철학을 담아, 덧셈, 뺄셈, 곱셈, 나눗셈의 기본적인 기능을 갖춘 계산기를 개발하였습니다.
